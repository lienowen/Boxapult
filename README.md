# Parcel Patrol

Parcel Patrol is a complete browser arcade loop built for fast CrazyGames validation. The player pilots a courier drone, protects a city delivery route, collects parcels, defeats hostile drones, builds a score combo, and survives a timed mission.

The GitHub repository keeps its original `Boxapult` name for now, but the game and package are Parcel Patrol.

## Current game loop

1. Open the title screen and start a delivery route.
2. Move vertically with pointer, touch, arrow keys, or `W` / `S`.
3. Fire with click, tap, or Space.
4. Collect parcels, destroy scout and raider drones, and protect three lives.
5. Complete the 60-second route or lose the drone.
6. Review score, deliveries, route grade, and best score.
7. Retry immediately or return to the menu.

## Technology

- Phaser 3.90
- TypeScript strict mode
- Vite
- Arcade Physics
- Vitest and an automated architecture boundary check
- Runtime-generated original vector artwork
- Runtime-generated Web Audio sound effects

## Open-source base

The scene organization and arcade shooter loop were adapted from the official Phaser demo **Phaser's Revenge**, distributed through `phaserjs/create-game` under the MIT license.

No original demo artwork, fonts, music, or sound assets are included. Parcel Patrol's current visuals and sounds are generated by code in this repository.

## Active scope

The current milestone is one polished, replayable route. Do not add a store, progression economy, multiple currencies, multiplayer, or advertisements until the title-to-result loop passes browser and mobile playtesting.
